# The-Gradient-Causal-Gap-When-Gradient-Magnitude-Misidentifies-Causally-Important-Components

Code for the paper submitted to TrustNLP @ ACL 2026.

## Overview

Formalizes the **Gradient-Causal Gap** — the divergence between gradient-based importance and causal importance in Transformers. Introduces **Hidden Heroes** (low gradient, high causal impact) and **Gradient Bloats** (high gradient, low causal impact).
